redis学习

发表于 2018-01-29 |

数据类型

  • string
  • hash
  • list
  • set
  • zset

redis命令

redis命令

  • 远程连接 redis-cli -h host -p port -a password

redis键（key）

命令 含义
del key 删除
exists key 判断是否存在

redis字符串（string）

命令 含义
del key 删除值
get key 获取值
set key value 设置值

redis哈希（hash）

命令 含义
hdel key field 删除哈希表字段
hmget key field 获取值
hmset key field value 设置值

redis列表（list）

命令 含义
lpush key value 将一个或多个值插入到列表头部
rpush key value 在列表中添加一个或多个值
lrange key start end 设置值

redis集合（set）

命令 含义
sadd key member 向集合添加一个或多个成员
smembers key 返回集合中的所有成员

redis有序集合（sorted set）

命令 含义
zadd key score member 向有序集合添加一个或多个成员，或者更新已存在成员的分数

redis事务

命令 含义
DISCARD 取消事务，放弃执行事务块内的所有命令。
EXEC 执行所有事务块内的命令
MULTI 标记一个事务块的开始
UNWATCH 取消 WATCH 命令对所有 key 的监视
WATCH key [key …] 监视一个(或多个) key ，如果在事务执行之前这个(或这些) key 被其他命令所改动，那么事务将被打断。
